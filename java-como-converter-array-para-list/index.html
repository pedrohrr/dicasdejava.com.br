Java: Como converter Array para List

22 de junho de 2017
Por Gustavo Furtado de Oliveira Alves

Para converter um Array (Vetor) em um List, nós podemos utilizar a classe java.util.Arrays que é nativa da API java.

Veja no exemplo abaixo como converter um Array em um List.

package br.com.dicasdejava.util;

import java.util.Arrays;
import java.util.List;

public class ArrayParaList {

    public static void main(String[] args) {
        String[] nomes = new String[]{"Gustavo", "Maria", "José", "João", "Ana"};

        List listaDeNomes = Arrays.asList(nomes);

        System.out.println(listaDeNomes);
    }
}

Saída:

[Gustavo, Maria, José, João, Ana]
